학습용 데모 실제 TCP 실험실

실제 TCP 서버와 클라이언트를 실행하고 통신 기록을 남기기

이 페이지는 교육용 재현이 아니라 실제 Node TCP 서버와 실제 TCP 클라이언트를 실행한다. 브라우저는 제어 UI 역할만 하고, 실제 소켓은 서버 프로세스가 열며 모든 송수신 기록은 파일과 라이브 로그에 남는다.

실제 리스너

새 TCP 서버를 실제로 띄우고, 외부 프로그램이나 클라이언트가 접속한 모든 연결을 주소, 포트, 바이트 단위로 기록한다.

클라이언트

실제 소켓 연결

브라우저가 raw TCP를 직접 열 수 없으므로, 이 버튼은 서버 프로세스에 실제 TCP 클라이언트를 생성하도록 요청한다. 접속 후에는 어느 소켓에서든 데이터를 보내고 종료할 수 있다.

연결된 클라이언트가 아직 없다.

전송 / FIN / 강제 종료
